# Installation

You can choose either the Python or TypeScript version. Both provide the same `ddss` command-line interface.

## Python Version

### Using uvx (Recommended)
```bash
uvx ddss
```

### Using pip
```bash
pip install ddss
ddss
```

## TypeScript Version

### Using npx (Recommended)
```bash
npx @hzhangxyz/ddss
```

### Using npm
```bash
npm install -g @hzhangxyz/ddss
ddss
```
