# 项目简介

分布式演绎系统工具集（DDSS）是一个具有可扩展架构的分布式演绎系统，在 **Python** 和 **TypeScript** 中均提供了一致的实现。目前支持包括前向链接（Forward-chaining）、E-graph 等在内的分布式引擎。

## 设计哲学

DDSS 采用了模块化架构，将演绎系统分解为独立但协作的子系统：

1. **关注点分离**：每个模块专注于特定的推理任务。
2. **并发执行**：所有模块通过共享数据库异步协作，充分利用系统资源。
3. **持久化存储**：使用数据库存储事实和想法，确保数据一致性。

### 核心枢纽逻辑

系统以数据库为中心，通过两个表（`facts` 和 `ideas`）实现子系统间的交互：

- **急切引擎**（如前向链接）：读取事实并积极产生新事实。它们还会添加“想法”（Ideas）来广播“我需要 XXX”——指示它们需要哪些新事实来产生更多结果。

- **延迟引擎**（如 E-graph）：如果过于急切可能会产生过多事实，因此它们静默地接受事实而不产生太多新事实。只有当它们看到其他引擎发出的、且它们能够（部分）实现的“想法”时，才会产生事实。
